A modern web browser with JavaScript enabled.

### Installation

1. Clone the repository:
   ```bash

   ```

2. Navigate to the project directory:
   ```bash

   ```

3. Open `index.html` in your browser or use a local server:
   ```bash

   # Using Node.js
   npx serve
   
   # Using PHP
   php -S localhost:8000
   ```
